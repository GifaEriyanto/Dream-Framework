Dream Framework

Dream is HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.

Download Dream

Grid

Dream Framework is available for responsive grid system. Default grid is using class .col-*. If you want to make custom view for all device, just give class .sm-* for mobile view, .md-* for tablet view, and .lg-* for large desktop resolution (> 1440px);

						<div class="col-*"></div>

Container

Container is class for make your website like in a box. The default value is 1140px. If you want to custom it, just add property container="width_value".

							<div class="container"></div>
							

							

							For Custom Width :
							

							

							<div class="container" container="960px"></div>

Row for Height

Row in here is value for height the element. Each row have 50px value for height and available from .row-1 until .row-20.

							<div class="row-1"></div>

Background

Below is code for make block with background image. Just type bg-image="image_path" and give class bg-fill for make image view better.

							<div class="bg-fill" bg-image="..."></div>

Example:

And below is code for make block with background color. Just type bg-color="your_color". This code is support all color types like hex, rgb, rgba, or everything value that available in CSS.

							<div  bg-color="..."></div>
